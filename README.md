# :memo: 项目简介
**VVChat** 是一个基于 Vue 3 和 Supabase 的实时匿名聊天室应用。所有消息在发送后 1分钟自动销毁，支持图片和附件发送，适合临时讨论、快速沟通和私密交流。
> VV = 微微 · Chat = 聊天
> > 让聊天更轻松，让隐私更安全
# :dart: 适用场景
- 临时项目讨论
- 朋友间快速沟通
- 匿名意见交流
- 阅后即焚的私密对话
# :sparkles: 功能特点
## :speech_balloon: 实时聊天
- 基于 Supabase Realtime 的消息实时同步
- 实时显示在线用户数量
- 消息发送状态提示（发送中 :heavy_check_mark: / 成功 :heavy_check_mark: / 失败 :x:）
- 支持消息重发
## :fire: 阅后即焚
- 所有消息发送后 1分钟自动销毁
- 消息倒计时实时显示
- 即将过期消息高亮提示（最后10秒）
- 过期消息自动灰化删除
## :framed_picture: 图片分享
- 支持多图上传（最多9张）
- 自动图片压缩优化（5MB以内）
- 图片网格预览（1-4张自适应布局）
- 大图灯箱查看（支持左右切换）
- 图片懒加载
## :paperclip: 附件发送
<details>
<summary>支持多种文件格式</summary>

  - :page_facing_up: PDF、Word、Excel、PowerPoint
  - :memo: 文本文件、CSV、JSON
  - :card_index_dividers: ZIP、RAR、7z 压缩包

</details
  
  - 文件大小限制 10MB
  - （PDF/Word/Excel/图片/压缩包等）
  - 一键下载，下载计数统计
## :smile: 表情支持
- 内置表情选择器（30+常用表情）
- 支持表情搜索
- 快捷输入
## :iphone: 移动优先
- 完美适配手机端
- 侧边栏滑动菜单
- 触摸优化，流畅滚动
- 响应式布局（PC/平板/手机）
## :art: 界面美观
- 清爽的绿白配色
- 圆角设计，现代感强
- 平滑动画过渡
- 发送/接收消息气泡区分
